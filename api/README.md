# Store API

## Installation

`npm install` and you're good to go.

## How to Run

### Dependencies

`npm install` and you're good to go.

### Docker

- Build the image using `DOCKER_IMAGE_VERSION=CHOOSE_VERSION make docker/build` 
- `docker-compose up`

 *DON'T FORGET TO CHANGE THE VERSION ON docker-compose.yml`

### Local

- Run postgres `make postgres/up`
- Run npm start
